# 🧮 Calculator

## 📌 Description
This is a simple Calculator application developed using Python as part of the **CodeSoft Python Programming Internship**. The program performs basic arithmetic operations based on the user's choice.

## ✨ Features
- Addition
- Subtraction
- Multiplication
- Division
- Handles division by zero
- Simple menu-driven interface

## 🛠️ Technologies Used
- Python 3

## ▶️ How to Run
1. Clone this repository.
2. Open the project in your preferred Python IDE.
3. Run the Python file.
4. Enter two numbers and choose the desired operation.

## 📚 Concepts Used
- Variables
- User Input
- Conditional Statements (`if-elif-else`)
- Arithmetic Operators
